脚本功能

人人网自动在后台访问最近来访中的用户，当前回访进度会显示在最近来访框的右上角。

本脚本适用于那些对那些回访有强迫症，如果不回访会坐立不安，但是又压根不想看别人状态的人。

程序会对方在近三天来访过，且距离上次该脚本访问该用户的时间超过一天的情况下，自动访问对应用户。

使用

访问首页后其他均由脚本自动完成。

第一次使用或长期不使用后第一次使用可能需要很长的时间，如果您关闭了该页面下一次访问会接着这次访问到的最后一个人继续重新访问。

脚本只有在您打开人人首页之后才会运行和进行回访，不可能在您完全不登录人人网的情况下做任何事情。所以不要指望浏览器里面装完这个插件就能连人人网都不打开就自动地回访，最少也要登录一下嘛。

脚本权限和用户隐私

脚本中不包含收集个人信息的内容

本脚本在本地保存的信息包括：您的人人网帐号编号，您好友的人人网帐号编号，您好友最后一次访问您的时间以及本脚本最后一次访问您对应好友的时间。这些信息可以被任何可以使用本浏览器的用户以及任何可以在renren.com的域上运行的脚本所读取。本脚本未通过其他方式保存任何信息。

脚本会在页面加载好后自动拉取最近来访的记录，在需要回访的时候会在后台打开该用户的主页，脚本没有进行其他网络访问。

您如果需要临时禁用该功能请到“附加组件管理器”-“脚本管理”里面禁用该脚本。
