XiaoNei Timer

人人网计时器

安装脚本 错误反馈
在人人网顶部显示一个计时器
包括当日活动时间和累计时间
支持v6界面、v7界面和3g界面

珍爱生命，远离SNS

脚本功能

脚本会记录人人网页面被用户打开的时间。并将时间显示在页面上
只有人人网的页面是浏览器显示的当前页面，且浏览器为当前系统中活动的窗口时才会记录时间。

v6界面，当日活动时间会显示在导航栏，鼠标指向时会显示累计的时间。

v7界面，当日活动时间和累计活动时间会显示在侧栏（只支持首页）。

脚本权限和用户隐私

脚本使用了getValue，setValue用于存储计时器记录的时间。
getValue和setValue所存储的数据可以被脚本自己和其他有权限访问存储该数据的文件的用户所读写。
脚本不会进行网络访问，也不会记录其他数据。

已知支持的浏览器

  • Firefox + GreaseMonkey
  • Chrome + Tampermonkey
